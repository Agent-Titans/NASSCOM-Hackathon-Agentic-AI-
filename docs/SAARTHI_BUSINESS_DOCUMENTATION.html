Executive summary

SAARTHI (Sanskrit: companion, guide) automates the first mile of enterprise IT incident handling. Every ticket is sanitized, classified, matched against 1,000 resolved historical cases, routed to the correct department and Hand, and given actionable guidance — typically in under 20 seconds.

1. Problem & business impact

Enterprise IT helpdesks face password resets, VPN failures, application outages, database issues, and security incidents — often triaged manually with inconsistent results.

ChallengeImpact
Manual triage & mis-routingWrong queue → SLA breach, customer dissatisfaction
Inconsistent categorisationPoor analytics, repeated escalations
Tacit senior knowledgeSlow onboarding, tribal runbooks
Security under-triageCompliance and breach risk
High cost per ticketL1/L2 labour on repetitive sorting

SAARTHI reduces time-to-route from minutes–hours to seconds and deflects Hand 1 issues to self-service.

2. Solution — five-agent pipeline

Employee submits ticket → Guardrail (PII redaction, injection defence) → Retrieval (ChromaDB + 1,000 resolved tickets) → Classifier (Google Gemini 2.5 Flash) → Router (deterministic rules — no LLM) → Resolver (RAG-grounded or Gemini steps) → Supervisor (confidence + final Hand) → Employee portal / Agent queue / SecOps
AgentFunctionTechnology
GuardrailPII redaction, prompt-injection scanRegex + Gemini security scan
RetrievalSimilar resolved ticketsChromaDB + keywords + Gemini embeddings
ClassifierCategory & confidenceGemini 2.5 Flash (primary)
RouterHand + departmentrouting_rules.json (O(1))
ResolverResolution stepsRAG match or Gemini generate
SupervisorFinal Hand + bandWeighted c_total formula

5. RAG, Gemini embeddings & fallbacks

Retrieval-Augmented Generation

  • 1,000 resolved tickets indexed in ChromaDB (local MiniLM default; Gemini embedding optional)
  • Similarity gate — weak matches flagged as low grounding
  • Cache warming — background index + disk embedding cache on login

Graceful degradation

StagePrimaryFallback
ClassifyGemini JSONRAG neighbour → keyword index
Security"Security incident:" short-circuitSecOps Hand 3
ResolveRAG similar ticketGemini → heuristics → generic steps
EmbedGemini embedding-001Local vectors + keywords
HTTP 429/503Retry with backoffDegrade to cached / keyword paths
