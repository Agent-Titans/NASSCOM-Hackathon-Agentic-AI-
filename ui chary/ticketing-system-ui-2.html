3-Hand RAG Agent Flow
How the AI routes every ticket from submission to resolution
Hand 1 — High Confidence Match
RAG similarity score ≥ 0.85
Auto-Resolved
The agent found an exact or near-exact match in the knowledge base.
  • Resolution copied from matched ticket, posted as comment
  • Status set to Resolved immediately
  • Assigned back to the submitting user (user_id)
  • User can still comment if the resolution doesn't work
Hand 2 — Medium Confidence Match
RAG similarity score 0.50–0.84
Dept Assigned
A related but not identical issue was found.
  • High-level resolution steps from similar ticket posted as comment
  • Status remains Open
  • Assigned to the same department as the matched ticket
Hand 3 — No Match Found
RAG similarity score < 0.50
Triage
No similar issue in the knowledge base.
  • LLM-generated guidance posted as comment (clearly labelled AI-generated, unverified)
  • Note: "No previous issues found — this resolution is LLM-based"
  • Assigned to Triage Team
What happens next? The Triage team assigns the ticket back to you requesting more information. You can then submit a new ticket with the additional details — the same 3-hand flow runs again.
